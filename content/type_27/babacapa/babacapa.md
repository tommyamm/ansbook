### №18677 на КЕГЭ

**Задача в двух словах:**  
В файле находятся координаты звёзд ($x$, $y$). Нужно разделить их на кластеры (группы), найти центр каждого кластера, а потом посчитать среднее положение всех центров и вывести его с большой точностью.

#### Что такое кластер и центр кластера?
- **Кластер** — это группа «соседних» звёзд (не меньше 30 штук).
- Каждая звезда принадлежит только одному кластеру.
- **Центр кластера (центроид)** — это та звезда внутри кластера, от которой сумма расстояний до всех остальных звёзд этого кластера **самая маленькая**.

Расстояние между двумя точками считается по обычной формуле:
$$
d = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}
$$

**Важно:**  
В файле **B** есть 3 «лишние» точки — аномалии (помехи). Их нужно удалить, потому что они не относятся ни к одному кластеру.

---

### План решения

1. **Прочитать данные** из файла.
2. **Удалить аномалии** (выбросы).
3. **Разделить оставшиеся точки на кластеры** (для файла B — на 3 кластера).
4. **Найти центр каждого кластера**.
5. **Посчитать среднее арифметическое** координат всех центров ($P_x$ и $P_y$).
6. **Умножить** каждое на 100000, взять целую часть и вывести.

### Вспомогательные функции

```python
def plot(data, color='blue'):
    x_values = [point[x] for point in data]
    y_values = [point[y] for point in data]
    plt.scatter(x_values, y_values, color=color)
```

Вычисляет евклидово расстояние между двумя точками.
```python
def dist(point_a, point_b):
    return (
        (point_b[x] - point_a[x]) ** 2 +
        (point_b[y] - point_a[y]) ** 2
    ) ** 0.5
```

Удаляет аномалии (выбросы) по трём прямым линиям.
```python
def clear_data(data):
    clear = []
    for point in data:
        if point[y] > point[x] + 5:      # выше верхней границы
            continue
        if point[y] > -point[x] + 20:    # выше второй границы
            continue
        if point[y] < point[x] - 12:     # ниже нижней границы
            continue
        clear.append(point)
    return clear
```

Разделяет точки на 3 кластера с помощью прямых линий.
```python
def clusterization(data):
    clasters = [[], [], []]   # три пустых списка для кластеров
    
    for point in data:
        if point[y] < -1.42 * point[x] + 10.856:          # первая разделяющая прямая
            clasters[0].append(point)
        elif point[y] < -1.223 * point[x] + 15.009 and point[x] < 9:  # вторая прямая + ограничение по x
            clasters[1].append(point)
        else:
            clasters[2].append(point)
    
    return clasters
```

Находит центр кластера — точку, ближайшую ко всем остальным.
```python
def get_center(claster):
    center = None
    min_sum = float('inf')
    
    for point in claster:
        # Сумма расстояний от текущей точки до всех остальных в кластере
        sum_ = sum(dist(point, other) for other in claster)
        
        if sum_ < min_sum:
            min_sum = sum_
            center = point
            
    return center
```

### Основной код решения (для файла B)

```python
import matplotlib.pyplot as plt

# ====================== Основная программа ======================

# Читаем данные из файла
with open("27B_18677.txt") as file:
    first_line = file.readline()          # пропускаем заголовок
    data = [list(map(float, i.replace(",", ".").split())) for i in file.readlines()]

x, y = 0, 1   # индексы координат

# 1. Удаляем аномалии (лишние точки)
data = clear_data(data)

# 2. Разделяем точки на три кластера
claster_1, claster_2, claster_3 = clusterization(data)

# 3. Рисуем кластеры разными цветами (для проверки)
plot(claster_1, 'red')
plot(claster_2, 'black')

cent_1 = get_center(claster_1)
cent_2 = get_center(claster_2)
cent_3 = get_center(claster_3)

# 5. Считаем среднее положение центров
p_x = abs(int(((cent_1[x] + cent_2[x] + cent_3[x]) / 3) * 100_000))
p_y = abs(int(((cent_1[y] + cent_2[y] + cent_3[y]) / 3) * 100_000))

print(p_x, p_y)

# 6. Показываем график
plt.grid(True)
plt.show()
```

- Вторая прямая: $Y = -1.223 * X + 15.009$

Границы для удаления аномалий:
- $Y = 1.0 * X + 5$
- $Y = -1.0 * X + 20$
- $Y = 1.0 * X - 12$

Эти прямые хорошо отделяют кластеры друг от друга и отбрасывают лишние точки.
