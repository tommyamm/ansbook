### №27637 на КЕГЭ

**Что нужно сделать в этой задаче:**

В файле даны координаты точек (звёзд, объектов и т.д.) на плоскости. Все точки делятся на **два кластера**.  
Твоя задача:
1. Разделить все точки на два кластера.
2. Найти **центр** каждого кластера (особую точку внутри кластера).
3. Найти одну **«странную» точку** (она дана в условии задачи).
4. Посчитать сумму расстояний от центров кластеров до этой странной точки.
5. Умножить результат на $10000$ и вывести целую часть.

Также в задаче просят вывести **минимальное количество точек** в одном из кластеров.

### Разбор решения

```python
with open("27_A_27637.txt") as file:
    data = [list(map(float, i.replace(",", ".").split())) for i in file.readlines()]

x, y = 0, 1        # индексы: 0 — координата x, 1 — координата y
```

Сначала читаем все точки из файла. Каждая строка — это одна точка `[x, y]`.

---

#### 1. Разбиение на два кластера

```python
cl1 = []
cl2 = []

# Разбиение на кластеры вручную через горизонтальную линию
for point in data:
    if point[y] > 15:      # если y больше 15
        cl1.append(point)  # -> первый кластер (верхний)
    else:
        cl2.append(point)  # -> второй кластер (нижний)
```

**Почему так просто?**  
В этой задаче кластеры хорошо разделяются **горизонтальной прямой** $y = 15$.  
Все точки выше $y=15$ попадают в один кластер, все ниже — в другой.  
Это самый простой способ кластеризации в данной задаче.

```python
print(min(len(cl1), len(cl2)))
```

Выводим, в каком кластере меньше всего точек (это часто спрашивают в задаче).

---

#### 2. Функция расстояния

```python
def dist(p1, p2):
    """Евклидово расстояние между двумя точками"""
    return ((p1[x] - p2[x]) ** 2 + (p1[y] - p2[y]) ** 2) ** (1/2)
```

Обычная формула расстояния между точками. Мы будем использовать её много раз.

---

#### 3. Функция поиска центра кластера

```python
def get_center(claster):
    """Находим центр кластера — точку, от которой сумма расстояний до всех остальных точек минимальна"""
    center = None
    min_sum = float("inf")
    
    for point in claster:
        # Считаем сумму расстояний от текущей точки до всех остальных
        cur_sum = sum(dist(point, other) for other in claster)
        
        if cur_sum < min_sum:
            min_sum = cur_sum
            center = point
            
    return center
```

**Что такое центр кластера?**  
Это не обязательно геометрический центр. Это та точка из кластера, которая «ближе всего» ко всем остальным точкам этого кластера (по сумме расстояний).

---

#### 4. Основная часть решения

```python
strange_point = [-1.0, 1.3]        # странная (заданная) точка

center_cl1 = get_center(cl1)
center_cl2 = get_center(cl2)

# Считаем сумму расстояний от обоих центров до strange_point
result = dist(center_cl1, strange_point) + dist(center_cl2, strange_point)

print(int(result * 10_000))        # умножаем на 10000 и берём целую часть
```

### Полный код

```python

with open("27_A_27637.txt") as file:
    # Читаем все точки из файла
    data = [list(map(float, i.replace(",", ".").split())) for i in file.readlines()]

x, y = 0, 1   # координаты x и y

# ------------------- 1. Разделяем точки на два кластера -------------------
cl1 = []   # верхний кластер
cl2 = []   # нижний кластер

for point in data:
    if point[y] > 15:           # горизонтальная линия y = 15 отлично разделяет кластеры
        cl1.append(point)
    else:
        cl2.append(point)

# Выводим размер меньшего кластера
print("Количество точек в меньшем кластере:", min(len(cl1), len(cl2)))

# ------------------- 2. Функции -------------------

def dist(p1, p2):
    """Расстояние между двумя точками"""
    return ((p1[x] - p2[x]) ** 2 + (p1[y] - p2[y]) ** 2) ** 0.5


def get_center(claster):
    """Находит центр кластера — точку с минимальной суммой расстояний до остальных"""
    center = None
    min_sum = float("inf")
    
    for point in claster:
        cur_sum = sum(dist(point, other) for other in claster)
        if cur_sum < min_sum:
            min_sum = cur_sum
            center = point
    
    return center


# ------------------- 3. Основные вычисления -------------------

strange_point = [-1.0, 1.3]   # заданная странная точка

center_cl1 = get_center(cl1)
center_cl2 = get_center(cl2)

# Сумма расстояний от центров кластеров до strange_point
total_distance = dist(center_cl1, strange_point) + dist(center_cl2, strange_point)

# Ответ: умножаем на 10000 и берём целую часть
print("Ответ:", int(total_distance * 10_000))
```

---

### Пояснения для понимания

- Кластеризация здесь очень простая — достаточно одной горизонтальной линии `y > 15`.
- Центр кластера ищется перебором: для каждой точки считаем сумму расстояний до всех остальных и выбираем самую «центральную».
- `strange_point = [-1.0, 1.3]` — это специальная точка, заданная в условии задачи.
- В ответе нужно вывести целую часть произведения суммы расстояний на 10000.
